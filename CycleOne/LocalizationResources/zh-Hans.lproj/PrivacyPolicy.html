隐私优先，无云端。

CycleOne 的唯一目标是保护你的周期数据，不让任何人获取，包括我们自己。

1. 仅本地存储

你的所有数据（日期、症状、心情）都仅通过 Core Data 存储在你的 iPhone 上。我们没有任何服务器，也不使用 iCloud 同步。

2. 无需账号

你无需注册或提供邮箱。你的数据绑定在设备上，而非身份账号。

3. 无广告追踪与分析

我们不使用任何第三方 SDK、广告追踪器或分析工具。我们无法知道你多久使用一次应用，也不知道你记录了什么。

4. 数据由你掌控

你可以随时将数据导出为 CSV 文件。你也可以在应用内删除所有数据，或通过卸载应用彻底清除数据。

5. 第三方服务

CycleOne 不连接任何第三方服务。应用不会发起网络请求，对外部服务器或 API 的依赖为零。

6. 数据保留

你的数据仅在应用安装期间存在。卸载 CycleOne 会永久移除所有数据，我们无法为你恢复。

7. 儿童隐私

CycleOne 不面向 13 岁以下儿童。我们不会有意收集儿童的个人数据。

8. 联系方式

如对本政策有疑问，请通过我们的 GitHub 仓库联系：github.com/VoxDroid/CycleOne
