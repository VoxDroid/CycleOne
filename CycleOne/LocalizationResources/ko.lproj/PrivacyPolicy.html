개인정보 우선. 클라우드 없음.

CycleOne은 단 하나의 목표로 만들어졌습니다. 우리를 포함해 누구에게도 주기 데이터가 노출되지 않도록 보호하는 것입니다.

1. 로컬 저장만 사용

모든 데이터(날짜, 증상, 기분)는 Core Data를 통해 사용자 iPhone에만 저장됩니다. 별도 서버를 운영하지 않으며 iCloud 동기화도 사용하지 않습니다.

2. 계정 불필요

회원가입이나 이메일 입력이 필요 없습니다. 데이터는 개인 계정이 아닌 기기에 연결됩니다.

3. 광고 추적 및 분석 없음

제3자 SDK, 광고 추적기, 분석 도구를 사용하지 않습니다. 앱 사용 빈도나 기록 내용도 수집하지 않습니다.

4. 데이터는 사용자 소유

언제든지 CSV 파일로 데이터를 내보낼 수 있습니다. 앱 내에서 모든 데이터를 삭제하거나 앱 삭제로 완전히 제거할 수 있습니다.

5. 제3자 서비스 미연동

CycleOne은 어떤 제3자 서비스에도 연결되지 않습니다. 앱은 네트워크 요청을 보내지 않으며 외부 서버/ API 의존성이 없습니다.

6. 데이터 보관 기간

데이터는 앱이 설치되어 있는 동안에만 유지됩니다. CycleOne을 삭제하면 모든 데이터가 영구적으로 제거되며 복구할 수 없습니다.

7. 아동 개인정보

CycleOne은 만 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.
