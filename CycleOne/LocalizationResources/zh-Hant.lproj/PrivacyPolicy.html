隱私優先，無雲端。

CycleOne 的唯一目標是保護你的週期資料，不讓任何人取得，包括我們自己。

1. 僅本地儲存

你的所有資料（日期、症狀、心情）都僅透過 Core Data 儲存在你的 iPhone 上。我們沒有任何伺服器，也不使用 iCloud 同步。

2. 無需帳號

你無需註冊或提供電子郵件。你的資料綁定在裝置上，而非身份帳號。

3. 無廣告追蹤與分析

我們不使用任何第三方 SDK、廣告追蹤器或分析工具。我們無法知道你多久使用一次應用，也不知道你記錄了什麼。

4. 資料由你掌控

你可以隨時將資料匯出為 CSV 檔案。你也可以在應用內刪除所有資料，或透過解除安裝應用徹底清除資料。

5. 第三方服務

CycleOne 不連接任何第三方服務。應用不會發起網路請求，對外部伺服器或 API 的依賴為零。

6. 資料保留

你的資料僅在應用安裝期間存在。解除安裝 CycleOne 會永久移除所有資料，我們無法為你復原。

7. 兒童隱私

CycleOne 不面向 13 歲以下兒童。我們不會有意蒐集兒童的個人資料。

8. 聯絡方式

如對本政策有疑問，請透過我們的 GitHub 儲存庫聯絡：github.com/VoxDroid/CycleOne
