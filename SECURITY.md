# Security Policy

## Supported Versions

We currently support the latest release of CycleOne.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✓         |
| < 1.0   | ✗         |

## Reporting a Vulnerability

As CycleOne is a privacy-focused application, we take security very seriously.

If you discover a security vulnerability within CycleOne, please open a GitHub Issue or contact the maintainers directly.

**Note**: Since CycleOne is 100% local-only with no server-side components, most "vulnerabilities" would involve local data access on a compromised device. However, we still value reports on encryption improvements or data leakage via system logs/backups.
